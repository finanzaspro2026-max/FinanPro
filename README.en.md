🛠️ Secure Installation Guide
Because FinanPro is independently developed and not distributed through commercial stores that charge high commissions, operating systems may show standard warnings:

###📱 For Android (APK)
Download the FinanPro-Android.apk file to your phone.
When opening it, your browser (Chrome or Firefox) will request permission to "Install apps from unknown sources".
Enable the permission in the settings pop‑up.
Tap "Install anyway" on the Google Play Protect warning.

###💻 For Windows (EXE)
Download and run the FinanPro-Setup.exe installer.
If Windows SmartScreen shows "Windows protected your PC", click the "More info" link.
Click the "Run anyway" button to start the installation wizard.
