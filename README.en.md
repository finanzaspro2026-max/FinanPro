🛠️ Secure Installation Guide
Because FinanPro is independently developed and not distributed through commercial app stores (which charge high commissions), operating systems may show standard warnings.

📱 For Android (APK)
Download the FinanPro-Android.apk file to your phone.
When opening it, your browser (Chrome or Firefox) will request permission to “Install apps from unknown sources.”
Enable the permission in the settings pop‑up.
Tap Install anyway on the Google Play Protect warning.
💻 For Windows (EXE)
Download and run FinanPro-Setup.exe.
If Windows SmartScreen shows "Windows protected your PC", click More info.
Click Run anyway to start the installer wizard.
